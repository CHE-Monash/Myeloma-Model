# Monash Myeloma Model — Economic Inputs

Reference for the cost and utility parameters the model uses to turn a simulated treatment pathway into
costs and quality-adjusted life years (QALYs). All of the economic logic lives in **`core/process_data.do`**
(the `process_data` program), which runs at the end of every simulation pass. This document lists the
parameter values, how they are applied, and how discounting works.

- **Currency:** Australian dollars (AUD).
- **Perspective / price basis:** the **Australian health system** — drug costs are the full PBS **DPMQ**
  (Dispensed Price for Maximum Quantity; patient co-payment **not** netted). `process_data.do` now **reads**
  the per-cycle drug and non-drug costs from `prep/inputs/treatment_costs_<year>.csv` (built by
  `prep/treatment_costs.do`, below) — they are no longer hardcoded. The `$cost_year` global **selects the
  cost file** (`treatment_costs_${cost_year}.csv`); if that year is absent, the latest available file is used
  with a note. Drug acquisition costs are PBS-based (dated extract); non-treatment (health-state) costs are
  phase-based from **Yap 2025** (MM-attributable, NSW linked data); ASCT from AR-DRG V11.0 × NEP.
- **Discount rate:** `$drate` (default **0.05** = 5% per year, the PBAC reference rate).

> ✓ The drug/regimen and non-drug costs are **derived** by **`prep/treatment_costs.do`** from committed PBS
> input tables in `prep/inputs/`, and `process_data.do` **reads the result** — the reproducible, versioned
> replacement for the old cost spreadsheet (no values are hardcoded any more). Utilities (`u*`) are still
> edited directly in `process_data.do`.

## Deriving and repricing costs

Two scripts, both runnable in Stata batch:

1. **`prep/extract_pbs_costs.do`** filters a dated full PBS Schedule extract (git-ignored raw source in
   `data/`, path `$pbs_src`; re-downloadable and swappable for the PBS REST API) down to the committed price
   inputs: `pbs_prices.csv` (AEMP per drug×strength×program×pack, cheapest brand — captures generic price
   disclosure), `pbs_fees.csv`, `pbs_markups.csv`, `pbs_copayments.csv`.
2. **`prep/treatment_costs.do [year] [oral_policy] [net_copay]`** builds the per-cycle drug costs and the
   (inflated) non-drug costs from:

| File | Role |
|---|---|
| `prep/inputs/treatment_regimens.csv` | Dosing spec (stable): per regimen×drug — dose, basis (`/m2`,`/kg`,`flat`), schedule, cycles, and any admin/vial overrides. |
| `prep/inputs/pbs_prices.csv` + `pbs_fees.csv` + `pbs_markups.csv` + `pbs_copayments.csv` | PBS ingredients (AEMP, fees, mark-up bands, co-payments) from a dated extract, via `extract_pbs_costs.do`. |
| `prep/inputs/other_costs.csv` | Non-drug costs, raw with **source year**: hospital/community/emergency by phase from Yap 2025 (myeloma, 2019 AUD); ASCT from AR-DRG V11.0 × NEP (2024-25). See *Non-treatment costs* below. |
| `prep/inputs/cost_index.csv` | Year × **ABS CPI All groups index** (series A2325846C), inflates the non-drug costs (`index[target]/index[source]`, latest year ≤ target carried forward). Generated by `prep/build_cost_index.do`. Drug prices are **not** inflated — they use the actual dated PBS values. |

**DPMQ method** (perspective = Australian health system: full DPMQ, co-payment not netted):
- **EFC injectables** (bortezomib, carfilzomib, daratumumab-IV): fewest vials (min count, then min AEMP) to
  cover the dose + EFC preparation fee, per administration.
- **Section 100 HSD orals** (lenalidomide, pomalidomide, thalidomide): pack AEMP + dispensing fee.
- **General Schedule orals** (dexamethasone, oral cyclophosphamide): pack AEMP + wholesale + pharmacy mark-up
  + dispensing fee.
- **Oral pack policy** (arg 2): **`wholepack` (default, PBAC convention — dispensing wastage costed)** or
  `prorata` (units consumed only, a sensitivity). Injectables are always fewest-vials.
- **`net_copay`** (arg 3): `0` = full DPMQ (default); `1` = net the co-payment (concessional default,
  switchable to general) for a cost-to-government view.

Output: `prep/inputs/treatment_costs_<year>.csv` (the `c*` per-cycle drug costs + inflated non-drug costs).
The script **validates** four externally-verifiable published DPMQs (carfilzomib $2,503.54, bortezomib
$113.03, daratumumab $7,307.30, lenalidomide $778.73) before writing. To reprice for a new schedule: point
`$pbs_src` at a new dated PBS extract and re-run `extract_pbs_costs.do`; add the new June CPI value to
`prep/build_cost_index.do` and re-run it; then run `prep/treatment_costs.do <year>`.

> ✓ **Kd resolved.** The old hardcoded `cKd = 15025` costed the escalated carfilzomib dose (56 mg/m² ≈ 108 mg)
> as **one** 60 mg vial; the first-principles build correctly uses **two** (2×60 mg + prep = $2,503.54 /
> administration). The 2026 per-vial price roughly halved vs the spreadsheet, so the correct 2026 **cKd is
> ≈ 12,630 / cycle** — *below* the old 15,025 despite fixing the vial undercount. (The earlier "2 vials ×
> DPMA ≈ 25,028" spreadsheet fix double-counted and is superseded.)

## Discounting

Costs and QALYs are discounted with a **continuous (uniform-over-interval) approximation**. A stream that
accrues a total undiscounted amount `X` uniformly over an interval `[t0, t1]` (times in months) has
discounted value

```
X · [ (1+r)^(−t0/12) − (1+r)^(−t1/12) ] / ( ln(1+r) · (t1−t0)/12 )
```

where `r = $drate`. One-off costs (ASCT) are discounted at the event time: `X · (1+r)^(−t/12)`. Discounting
uses **relative-time markers** (`TSD_*_ref`): for a from-diagnosis run (`$line == 1`) time 0 is diagnosis;
for a line-`L` entry run (`$line > 1`) time 0 is the start of line `L`.

## Treatment (drug) costs

Each regimen has a **unit cost per cycle** (or per course) applied over the simulated treatment duration
`TXD_L*` (months). `TXD × 30.4375 / cycle_days` converts months to number of cycles. The bortezomib-based
induction regimens are **capped** at a fixed number of cycles (fixed-duration induction); the others accrue
per cycle for the whole `TXD`.

Unit costs below are the **2026 PBS DPMQ, whole-pack** values from `treatment_costs_2026.csv` (the
`c*` locals are read from that file; the formula structure — cycle length, caps — is unchanged).

| Regimen | TXR code | Unit cost (AUD) | Cycle | Cap | Applied as |
|---|---|---|---|---|---|
| VCd (Bort/Cyclo/Dexa) | 4 | 663.69 | 21 d | 4 cycles | `cVCd · min(4, TXD·30.4375/21)` |
| VRd (Bort/Lena/Dexa) | 31 | 1,013.23 | 21 d | 5 cycles | `cVRd · min(5, TXD·30.4375/21)` |
| Vd (Bort/Dexa) | 5 | 491.00 | 21 d | 8 cycles | `cVd · min(8, TXD·30.4375/21)` |
| Rd (Lena/Dexa) | 7 | 817.61 | 28 d | — | `cRd · TXD·30.4375/28` |
| Kd (Carf/Dexa) | 49 | 12,629.70 | 28 d | — | `cKd · TXD·30.4375/28` |
| DVd (Dara/Bort/Dexa) | 80 | 12,345.92 | 28 d | — | `cDVd · TXD·30.4375/28` |

### How each per-cycle regimen cost is built (PBS DPMQ)

Every regimen cost above is built from first principles as the PBS **DPMQ** (Dispensed Price for Maximum
Quantity) of its component drugs, summed over the doses in one cycle. The general PBS pricing identity is:

```
DPMQ = AEMP (× units dispensed) + wholesale mark-up + pharmacy mark-up + fees
```

where **AEMP** = Approved Ex-Manufacturer Price (the manufacturer's price the Government starts from). The
model takes AEMP as the **cheapest brand** at each strength/pack from a dated PBS Schedule extract (this is
what captures generic price disclosure). Which mark-ups and fees apply depends on the drug's **supply
channel** — the model's nine drugs fall into three, keyed off the PBS program code:

**(a) EFC injectables — bortezomib, carfilzomib, daratumumab (IV).** Program IN (public). Per
*administration*:
1. Convert the dose to mg: `dose × body size` (BSA 1.93 m², weight 81.1 kg, or flat — population averages).
2. **Fewest vials** to cover the dose — the minimum *number* of vials, then, among those, the minimum-AEMP
   combination (vial pricing is linear per mg for carfilzomib/daratumumab, so this = the smallest total mg
   ≥ dose). Any unused drug in a part-vial is real, unavoidable wastage and is costed.
3. `+ EFC preparation fee` **$92.56** per preparation (from `pbs_fees.csv`, program IN).
4. `DPMQ_admin = Σ(vial AEMP) + $92.56`, then `× administrations per cycle`.

**(b) Section 100 HSD orals — lenalidomide, pomalidomide, thalidomide.** Program HB (public). Per pack:
`pack AEMP + dispensing fee $9.24`. No wholesale/pharmacy mark-up (S100 drugs bypass community pharmacy).

**(c) General Schedule orals — dexamethasone, oral cyclophosphamide.** Program GE. Per pack: `pack AEMP +
wholesale mark-up + pharmacy mark-up + dispensing fee $9.24`, where the mark-ups are the banded
community-pharmacy (`s90-cp`) schedule in `pbs_markups.csv`:

| Mark-up | AEMP band | Amount |
|---|---|---|
| Wholesale | < $5.51 | fixed $0.24 |
| Wholesale | $5.51 – $1,000 | 4.30% × AEMP |
| Wholesale | $1,000 – $10,000 | 2.00% × (AEMP − 1,000) + $43 |
| Wholesale | > $10,000 | fixed $223 |
| Pharmacy | < $100 | fixed $5.12 |
| Pharmacy | $100 – $2,000 | 5.00% × (AEMP − 100) + $5.12 |
| Pharmacy | > $2,000 | fixed $100.12 |

**Oral pack policy** (`treatment_costs.do` arg 2): **`wholepack` (default)** dispenses whole packs per cycle
(`⌈units ÷ pack size⌉`), so pack wastage is costed — the **PBAC convention**; `prorata` costs only units
consumed (a sensitivity). Injectables are always fewest-vials.

**Perspective** is the Australian health system — the **full DPMQ** (co-payment *not* netted). Co-payments
(general $25, concessional $7.70) are extracted so a cost-to-government view (`net_copay = 1`) can be produced
on request, but the base case does not net them.

**Worked examples (2026 PBS):**

- *Carfilzomib, 56 mg/m² dose (EFC):* dose = 56 × 1.93 = 108 mg → cheapest 2-vial cover of {10, 30, 60 mg}
  = **2 × 60 mg** (2 × $1,205.49) + $92.56 prep = **$2,503.54** per administration (matches the published DPMQ).
- *Lenalidomide, Rd cycle (S100):* 25 mg daily, days 1–21 = 21 capsules → exactly one pack of 21 →
  $769.49 + $9.24 = **$778.73** per cycle.
- *Dexamethasone 4 mg (General Schedule):* AEMP $4.84 → wholesale $0.24 (AEMP < $5.51) + pharmacy $5.12
  (AEMP < $100) + dispensing $9.24 = **$19.44** per pack of 30.
- *Kd per cycle (roll-up):* 2 administrations at 20 mg/m² (39 mg → 1 × 60 mg vial + prep = $1,298.05) +
  4 at 56 mg/m² ($2,503.54) + 3 dexamethasone doses of 40 mg (30 tablets = 1 pack = $19.44) =
  2 × 1,298.05 + 4 × 2,503.54 + 19.44 = **$12,629.70** = `cKd`.

Per-cycle regimen cost = Σ over its drugs. "Other" (code 0) = mean(VTd, TCd, Td, Vd); maintenance `cMNT` =
MRDR usage-weighted blend of lenalidomide (R) and thalidomide (T). Body size is a population average, so
per-patient dose variation is not modelled (matters only at vial-rounding boundaries for the /m² and /kg
injectables). `treatment_costs.do` validates four externally-verifiable published DPMQs (carfilzomib
$2,503.54, bortezomib $113.03, daratumumab $7,307.30, lenalidomide $778.73) before writing the output.

## ASCT and maintenance (L1 only)

| Item | Local | Cost (AUD) | Applied when | Applied as |
|---|---|---|---|---|
| Autologous stem-cell transplant | `cASCT` | 41,723 | `SCT_L1 == 1` | one-off at L1 end |
| Maintenance therapy | `cMNT` | 620.15 | `MNT == 1` | per 28-day cycle over `TFI_L1`: `cMNT · TFI_L1·30.4375/28` |

**`cASCT` source:** AR-DRG V11.0 (IHACPA) autologous haematopoietic stem-cell transplant codes **R06A/B/C**,
separations-weighted average inlier price weight **6.4537** (weighted by national separations 341/432/425 for
Major/Intermediate/Minor complexity), × National Efficient Price **$6,465** (2024-25) = **$41,723**. Held in
`other_costs.csv` at source year 2025 (not CPI-inflated — it is already a current NEP figure).

> **Why this figure and not the one Yap cites.** Yap's discussion mentions an external 2009 NSW ASCT cost of
> $62,812 (inpatient *and* outpatient). That is **not** the figure Yap used in their own costing — they cost
> the transplant as an AR-DRG hospital admission mapped to the National Hospital Cost Data Collection, the
> same DRG approach used here. So our R06 × NEP figure **matches what is embedded in Yap's initial-phase
> hospital cost**, which is exactly the amount we net back out (below) — keeping the two consistent. The
> $62,812 also bundles outpatient services (mobilisation, apheresis, follow-up), some of which already sit in
> Yap's Medicare (`cMBS`) component, so adopting it would double-count. *Caveat:* our figure is the inpatient
> transplant admission only, so it may under-represent the full transplant episode (harvest/apheresis) — a
> possible future refinement.

## Non-treatment (health-state) costs — phase-based

Applied by **phase of care** over the survival window from the starting line, and kept **split by
component** so a cost breakdown can be reported. Phases are on the diagnosis clock: **initial** = first 12
months, **terminal** = last 12 months of life, **continuing** = the span between (Yap short-survivor rules:
<2 yr → last year terminal + remainder initial; <1 yr → all terminal). `process_data` costs only the window
`[w0, OC_TIME]` (w0 = `OC_TIME − OC_TIME_L`, = 0 at L1), each phase contributing `rate × months-of-overlap/12`,
discounted over its own sub-interval — computed per component (`cHosp` + `cMBS` + `cEmer`) and summed.

Component annual rates (AUD, 2025); each cell is a `<component>_<phase>` parameter in `treatment_costs_<yr>.csv`:

| Component (Yap source) | Initial | Continuing | Terminal |
|---|---|---|---|
| `cHosp` — admitted hospital (APDC) | 30,879 (net of ASCT) | 14,841 | 73,137 |
| `cMBS` — out-of-hospital Medicare (MBS) | 9,878 | 4,947 | 9,648 |
| `cEmer` — emergency (EDDC) | 1,225 | 588 | 2,978 |
| **Phase total** | **41,982** | **20,375** | **85,763** |

**Source.** **Yap et al. 2025** (*Leukemia Research*; 45-and-Up Study, NSW linked data), Supplementary
Table 6 — the **excess (MM-attributable)** cost per person by phase of care, in **2019 AUD**. "Excess" = the
mean for MM patients minus the mean for matched non-cancer controls, i.e. the cost *attributable to* myeloma
(not the patient's total healthcare spend). Yap supersedes the earlier Merollini 2022 all-cancer figures,
which were *total* (not attributable) and conditional means, and roughly 2× higher. Each phase is reported
split into four components (percentages of the phase total):

| Phase (Yap STable6) | Total/person (2019) | Hospital (APDC) | Medicare (MBS) | PBS (drugs) | Emergency (EDDC) |
|---|---|---|---|---|---|
| Initial (first year) | $62,040 | 53.4% | 12.9% | 32.2% | 1.6% |
| Continuing (per year) | $39,681 | 30.3% | 10.1% | 58.4% | 1.2% |
| Terminal (last year) | $96,503 | 61.4% | 8.1% | 28.0% | 2.5% |

**Derivation** (in `treatment_costs.do`, from raw values in `other_costs.csv`):

1. **Drop PBS.** The model costs drugs separately (PBS DPMQ), so Yap's PBS component is excluded to avoid
   double-counting. The remaining three components map to the model's buckets: **APDC → `cHosp`
   (admitted hospital), MBS → `cMBS` (out-of-hospital Medicare), EDDC → `cEmer` (emergency)**.
2. **Sum the non-PBS components per phase** (2019 AUD): initial $33,129 + $8,003 + $993 = **$42,125**;
   continuing $12,023 + $4,008 + $476 = **$16,507/yr**; terminal $59,253 + $7,817 + $2,413 = **$69,482**.
3. **Inflate 2019 → target** by the ABS CPI All groups index (series **A2325846C**; June 2019 = 114.8,
   June 2025 = 141.7 → factor **1.234**), via `cost_index.csv`.
4. **Net the ASCT out of the initial phase.** The transplant admission is inside Yap's initial-phase
   hospital figure for the ~24% who transplanted. Since the model costs ASCT separately and
   patient-specifically (`cASCT`), the initial phase is reduced by `asct_prev × cASCT`, where **`asct_prev`
   = 0.24** is Yap's ASCT prevalence (STable10, 125/520). Continuing and terminal need no correction — ASCT
   is a one-off in the first year.

Net result (2025 AUD; inflation factor = 141.7 ÷ 114.8 = 1.2343): `cNT_initial` = 42,125 × 1.2343 −
0.24 × 41,723 = **41,982**; `cNT_continuing` = 16,507 × 1.2343 = **20,375**; `cNT_terminal` =
69,482 × 1.2343 = **85,763** (AUD/yr).

**How the phases are applied over a lifetime.** Phases are defined on the diagnosis clock — **initial =
first 12 months after diagnosis, terminal = last 12 months of life, continuing = the span between** — with
Yap's short-survivor rules: survival < 2 years → last year terminal + remainder initial; < 1 year → all
terminal. `process_data` charges only the survival window from the starting line, `[w0, OC_TIME]` (w0 =
`OC_TIME − OC_TIME_L`, = 0 at L1), each phase contributing `rate × months-of-overlap ÷ 12`, discounted over
its own sub-interval. So a from-diagnosis run pays the initial rate for the first year, the continuing rate
through the middle, and the terminal rate for the last year of life; a line-`L`-entry run pays only for the
portion after line `L` begins.

*Worked example — patient surviving 90 months from diagnosis (L1 run):* initial (months 0–12) $41,982 +
continuing (months 12–78, 66 mo) 66/12 × $20,375 = $112,062 + terminal (months 78–90) $85,763 =
**$239,807** undiscounted. (The old flat Merollini rate would have charged $52,147 × 7.5 = $391,000.)

Raw phase×component values (2019) are in `prep/inputs/other_costs.csv`; the CPI index is in `cost_index.csv`,
generated by **`prep/build_cost_index.do`** (ABS A2325846C June values hardcoded — add a year and re-run to
reprice).

## Utilities (QALYs)

QALYs are `Σ (time_in_state / 12) · utility_state`. The model uses a small set of health-state utilities by
pathway phase:

| Health state | Local | Utility |
|---|---|---|
| Treatment-free interval (off treatment / remission) | `uTFI` | 0.72 |
| On first-line treatment (`TXD_L1`) | `uTXD_L1` | 0.63 |
| On second-line treatment (`TXD_L2`) | `uTXD_L2` | 0.67 |
| Post-second-line (everything after `TXD_L2`, and all of L3+) | `uPostL2` | 0.63 |

How they map onto the pathway depends on the starting line (`$line`):

- **`$line == 1` (from diagnosis):** pre-L1 TFI (`uTFI`) → on-L1 (`uTXD_L1`) → L1 TFI (`uTFI`) → on-L2
  (`uTXD_L2`) → everything after L2 (`uPostL2`).
- **`$line == 2`:** on-L2 (`uTXD_L2`) → post-L2 (`uPostL2`).
- **`$line >= 3`:** the whole run from line `L` uses `uPostL2` (0.63).

## Output variables

`process_data` writes cost and QALY variables to the per-patient dataset (all AUD / QALYs):

- **Costs:** `cost_tx_L<line>`, `cost_tx_asct`, `cost_tx_mnt`, `cost_tx` (total treatment), `cost_nt`
  (non-treatment), `cost_total`. Non-treatment is also stored **split by component** — `cost_nt_hosp`,
  `cost_nt_mbs`, `cost_nt_emer` (they sum to `cost_nt`) — so a breakdown can be reported without re-running.
  A **`_d` suffix marks the discounted** counterpart (e.g. `cost_total_d`, `cost_nt_hosp_d`).
- **QALYs:** `qaly_tfi_DN`, `qaly_txd_L1`, `qaly_tfi_L1`, `qaly_txd_L2`, `qaly_post_L2`, `qaly_total`
  (and their `_d` discounted forms), collapsing to `qaly_txd_L2 + qaly_post_L2` at `$line == 2` and a
  single `qaly_total` at `$line >= 3`.

The ICER machinery (e.g. `transport_dvd`) collapses `cost_total_d` and `qaly_total_d` by arm.

## Caveats & maintenance notes

- **Non-treatment costs are phase-based (initial/continuing/terminal) but not line- or response-resolved** —
  they vary by time from diagnosis and time to death (Yap phases), not by which line or BCR the patient is in.
- **Utilities are phase-based, not response-based** — BCR (response) does not directly modulate utility.
- **The `Other` (code 0) drug cost is a single blended value** covering all pooled regimens (mean of VTd,
  TCd, Td, Vd); maintenance is a usage-weighted blend of lenalidomide and thalidomide.
- **Body size is a population average** — drug dosing uses a fixed BSA (1.93 m²) and weight (81.1 kg), so
  per-patient dose variation isn't modelled. This matters only for the size-adjusted drugs (carfilzomib
  `/m2`, daratumumab `/kg`), and mostly at vial-rounding boundaries; per-patient dosing is a possible future
  extension.
- **Reprice/sensitivity:** for **drug** costs, re-run `prep/extract_pbs_costs.do` (new dated PBS schedule)
  then `prep/treatment_costs.do <year> [oral_policy] [net_copay]` and set `$cost_year` to that year; for
  **non-drug** costs, add the new June CPI value to `prep/build_cost_index.do` and re-run it, then re-run
  `treatment_costs.do` (see *Deriving and repricing costs* above); **utilities** are still edited directly in
  `process_data.do`. Confirm the price year and source citations against the analysis manuscript before publication.
- **Cost basis change (Jul 2026):** moving off the frozen spreadsheet **lowered** both cost sides — 2026
  PBS DPMQ cut regimen costs (generic price disclosure: bortezomib, thalidomide, lenalidomide, carfilzomib),
  and switching non-treatment costs from Merollini (total, all-cancer) to **Yap 2025** (MM-attributable,
  phase-based) roughly halved the health-state cost (mean `cost_nt` fell ~40%). All analyses that re-run now
  use these costs, so results differ from earlier hardcoded-cost runs; check any near-submission analysis
  (and its `$cost_year`) before publishing.
